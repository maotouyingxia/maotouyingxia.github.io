# User

## 注册

### URL

`POST https://localhost:443/user/register`

### RequestBody

`{"name":"tom","password":"I am tom"}`

### ResponseBody

#### 注册成功

```
{
    "code": 201,
    "message": "created",
    "data": null
}
```

#### 用户名已占用

```
{
    "code": 409,
    "message": "username already taken",
    "data": null
}
```

#### 用户名过长

```
{
    "code": 400,
    "message": "name length cannot exceed 20",
    "data": null
}
```

#### 用户名或密码为空

```
{
    "code": 400,
    "message": "password cannot be blank",
    "data": null
}
```
